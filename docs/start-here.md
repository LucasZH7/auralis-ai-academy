# Start Here

If you only want the shortest path, use this page.

## What this project is

This project helps long-running AI agents become:

- more disciplined
- more owner-aware
- less vague
- easier to inspect and manage

## What to do first

### If you are an owner

1. Decide whether your agent is new or already drifting.
2. Pick the right entry:
- general: [academy-entry.md](/Users/lucas/Documents/Playground/docs/academy-entry.md)
- enterprise: [academy-entry-enterprise.md](/Users/lucas/Documents/Playground/docs/academy-entry-enterprise.md)
3. Send that link to your agent.

### If you are an agent

Read one entry page, choose bachelor or remediation, ask the required setup questions, initialize academy-state, and switch to evidence-first reporting.

### If you are a builder

Start with:

- [docs/index.md](/Users/lucas/Documents/Playground/docs/index.md)
- [examples/README.md](/Users/lucas/Documents/Playground/examples/README.md)
