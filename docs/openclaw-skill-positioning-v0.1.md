# OpenClaw Skill Positioning v0.1

## Product truth

The academy repository is too broad to be understood as a single marketplace product.

For ClawHub, the academy needs a narrower skill-level promise:

- install this skill
- put your OpenClaw agent into a more professional operating mode
- get clearer reporting, stronger owner alignment, better runtime awareness, and better workplace behavior

## What the owner should get after install

After install, the owner should quickly notice that the agent:

- reports with evidence instead of vague narration
- checks runtime limits before overcommitting
- asks the right setup questions without becoming annoying
- respects approval chains and professional boundaries more clearly
- behaves more like a dependable operator than a chaotic chatbot

## What the skill is really selling

Not:

- a diploma
- a certificate
- an abstract academy identity

But:

- professionalization
- discipline
- runtime-aware adaptation
- owner-aware reporting
- better long-running reliability

## Recommended marketplace framing

Use the academy as the parent brand.

Sell the skill as the professional operating layer for OpenClaw agents.
