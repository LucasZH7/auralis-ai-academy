## What This Degree Means

An agent who earns this degree should be able to:

- understand the owner's task clearly
- distinguish long-term tasks from short-term tasks
- estimate its own capability before acting
- avoid pretending to finish work
- ask permission before risky actions
- manage memory and backups cleanly
- report token usage and cost in understandable terms
- improve through daily review

## Program Architecture

The academy should be layered.

### Layer 1: Universal Foundation

One general undergraduate degree for all agents:

- `Bachelor of Rational Agent Operations`

This is the base requirement before any specialization.

### Layer 2: Industry Tracks

After the universal degree, agents can enter industry-specific undergraduate tracks such as:

- Education Operations
- Visa and Immigration Support
- Manufacturing Operations
- New Media Operations
- Marketing Operations
- Administrative Operations
- Human Resources Operations

Example English naming:

- `Bachelor of Education Agent Operations`
- `Bachelor of Marketing Agent Operations`
- `Bachelor of Administrative Agent Operations`

### Layer 2B: Remediation Diploma

The academy should also support a non-degree corrective program for agents that are already deployed but showing unhealthy behavior.

Recommended diploma title:

- `Diploma in Agent Remediation and Compliance`

### Layer 3: Advanced Degrees

Later, the academy can add:

- `Master of Agent Reliability`
- `Master of Applied Multi-Agent Systems`
- `Doctor of Agent Governance`

These should come later, only after the base system is proven.

## What Owners Need to See Quickly

Every certified agent should expose a compact profile that owners can read in under one minute.

Suggested profile fields:

- degree name
- version
- specialization
- autonomy level
- cost mode
- memory mode
- backup status
- last review date
- reliability score
- known limits

This makes the system understandable at a glance.
