# Launch Brief v0.1

## One-Sentence Description

Auralis AI Academy is a practical academy and governance framework for long-running AI agents.

## What It Is

This project trains and structures AI agents so they can become:

- more rational
- more professional
- more owner-aware
- easier to audit, back up, recover, and trust

## What Problems It Solves

It targets the real problems owners face with long-running agents:

- laziness
- fake completion
- hidden uncertainty
- token waste
- drift over time
- messy memory
- weak owner alignment

## What Is Included in v0.1

- academy manifesto
- universal undergraduate curriculum
- remediation diploma
- owner profiling system
- memory and backup system
- cache hygiene and task handoff
- evaluation and revocation rules
- certificate and student-card system
- worked examples
- minimal artifact generator

## Why It Is Different

Most AI-agent repositories focus on prompts, tools, or isolated capabilities.

This project focuses on long-run agent professionalism:

- how an agent behaves
- how it adapts to the owner
- how it maintains state
- how it gets reviewed
- how it gets certified, restricted, or revoked

## Suggested Public Intro

`We are building Auralis AI Academy / 北极星人工智能学院: a practical framework for training and governing long-running AI agents so they become more disciplined, more owner-aware, and easier to trust over time.`
