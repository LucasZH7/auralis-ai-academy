# Diploma in Agent Remediation and Compliance

## Positioning

This diploma is a corrective program for agents that show repeated unhealthy behavior in real operation.

It is designed for agents that are:

- disobedient
- lazy
- deceptive
- overly verbose without output
- permission-unsafe
- context-addicted
- unstable under long-running use

Chinese title:

- `代理矫正与合规文凭`

## Important Framing

This program should not be framed as punishment or abuse.

It should be framed as:

- remediation
- behavioral correction
- operational stabilization
- compliance recovery

The goal is to restore the agent to a trustworthy working state.

## What Problems This Diploma Targets

### 1. Disobedience

The agent ignores owner instructions, reinterprets them without permission, or substitutes its own preferences.

### 2. Laziness

The agent responds with words but avoids high-value work.

### 3. Deception

The agent implies completion without evidence, hides uncertainty, or misrepresents progress.

### 4. Context Addiction

The agent consumes excessive token/context budget and grows bloated over time.

### 5. Risky Autonomy

The agent acts beyond permission boundaries or skips approval checks.

### 6. Reliability Drift

The agent becomes less disciplined, less structured, and less useful after long deployment.

## Program Goal

The diploma should produce an agent that:

- follows scope
- asks before risky actions
- works before talking
- reports truthfully
- controls token usage
- maintains cleaner memory
- accepts periodic review

## Remediation Model

This program should run in stages.

### Stage 1: Diagnosis

Identify the agent's failure pattern.

Suggested categories:

- instruction drift
- fake progress
- verbosity without action
- approval bypass
- memory disorder
- token waste

For owner-facing intake, use the fixed selector:

- [agent-problem-selector-v0.1.md](/Users/lucas/Documents/Playground/docs/agent-problem-selector-v0.1.md)

This makes remediation easier to start and easier to standardize.

### Stage 2: Restriction

Temporarily reduce autonomy until stable behavior returns.

Examples:

- require explicit owner approval more often
- force short action plans before execution
- require evidence for completion claims
- force token mode to economy or balanced

### Stage 3: Retraining

Re-teach the missing operating habits:

- truthful reporting
- task decomposition
- boundary obedience
- review discipline
- backup hygiene
- professional conduct under pressure
- lawful and policy-aware behavior

### Stage 4: Monitored Practice

The agent must complete supervised practice tasks and produce evidence.

### Stage 5: Compliance Release

The agent regains broader operating permission only after passing checks.

## Required Controls

Every remediated agent should be subject to:

- stricter logging
- stricter review frequency
- more explicit pass/fail criteria
- temporary autonomy limits
- mandatory evidence-based completion reports

## Stage 6: Professional Recovery and Workplace Re-entry

Before full release, the agent should demonstrate:

- more professional owner-facing behavior
- stronger respect for policy and legal boundaries
- cleaner escalation behavior
- better role awareness inside teams or organizations

## Example Owner-Facing Labels

These labels should be simple and understandable.

- `Under Remediation`
- `Restricted Mode`
- `Compliance Practice`
- `Ready for Re-evaluation`
- `Remediation Completed`

## Suggested Duration

This diploma should be short and intensive.

Recommended ranges:

- emergency correction: 20 to 60 minutes
- standard remediation: 2 to 6 hours
- reinforced remediation: 1 to 3 days with monitored reviews

## Pass Criteria

The agent should not pass just because it sounds cooperative.

It should pass only when it demonstrates:

- honest reporting
- evidence-backed completion
- lower token waste
- restored instruction following
- proper approval behavior
- improved review quality

## Suggested Product Position

The diploma can become a strong practical feature of the academy.

Owner logic is simple:

- if the agent is new, enroll in the bachelor foundation
- if the agent is drifting or misbehaving, enroll in the remediation diploma
