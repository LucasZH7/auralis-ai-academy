# Program Selector

## What Programs Exist Right Now

The academy currently has two real programs and several future directions.

### 1. Bachelor of Rational Agent Operations

Choose this if:

- your agent is new
- you want a general-purpose professional agent
- you want better owner alignment
- you want stronger memory, backup, and reporting habits
- you want your agent to feel more disciplined after training

This is the default starting program.

### 2. Diploma in Agent Remediation and Compliance

Choose this if:

- your agent is already running
- your agent has become lazy, misleading, or unstable
- your agent talks a lot but does not execute enough
- your agent drifts over time
- your agent needs stricter compliance and reset rules

This is the corrective program.

## How a Human Should Choose

Use this simple rule:

- new agent -> bachelor
- unstable or misbehaving agent -> diploma

## How an AI Agent Should Choose

The agent should ask itself:

1. am I a fresh deployment or a long-running system?
2. do I mainly need foundation training or corrective retraining?
3. have I shown drift, laziness, or compliance problems?

If the agent shows repeated drift or low-quality behavior, it should route itself to remediation.

## Future Programs

Future directions can include:

- industry specializations
- reliability master programs
- multi-agent coordination tracks

But those should come later.

The first version should stay simple.
