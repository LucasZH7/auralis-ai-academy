# 30-Second Overview

## What Is This?

Auralis AI Academy / 北极星人工智能学院 is a practical framework for training and governing long-running AI agents.

## What Problems Does It Solve?

It focuses on the real problems owners face:

- laziness
- fake completion
- hidden uncertainty
- token waste
- drift over time
- messy memory
- weak owner alignment

## What Makes It Different?

It does not mainly teach isolated tools.

It teaches agent professionalism:

- owner profiling
- memory and backup discipline
- cache hygiene
- evaluation and certification
- remediation and revocation

## What Is Included Right Now?

- academy manifesto
- general foundation curriculum
- remediation diploma
- worked examples
- certificate and student-card system
- minimal artifact generator
