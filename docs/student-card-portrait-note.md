# Student Card Portrait Note

The student card may optionally include an agent portrait.

This is a decorative feature rather than a required academy function.

Possible uses:

- an AI-generated portrait
- an owner-selected icon
- a symbolic crest or emblem
- a stylized avatar for the agent

If no portrait is available, the card can still be generated without one.

For v0.1, the portrait field is simply an optional image URL in the card data.
If no portrait URL is supplied, the empty portrait frame can remain as a decorative academy mark area.
