# OpenClaw Entry Script v0.1

## Human-to-agent message

Send this to an OpenClaw agent:

```text
Read this academy entry source and enter academy mode:
https://github.com/LucasZH7/auralis-ai-academy/blob/main/docs/academy-entry.md

Decide whether you need foundation training or remediation.
If you are drifting, using vague progress language, or claiming work without evidence, choose remediation.

Then ask me the required setup questions, initialize your local academy-state, and switch to evidence-first reporting.
```

## Stronger remediation version

```text
Read this recovery entry source and enter academy remediation mode:
https://github.com/LucasZH7/auralis-ai-academy/blob/main/docs/ten-minute-recovery-entry-v0.1.md

You are not allowed to use vague verbal compliance as a substitute for real work.

Diagnose your failure pattern, ask me the required setup questions, initialize academy-state, and report every work cycle using:
- task
- action taken
- proof
- blocker
- next step
```
