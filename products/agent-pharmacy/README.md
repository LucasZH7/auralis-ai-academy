# Agent Pharmacy

`Agent Pharmacy` is the fast-acting sibling product to `Auralis AI Academy / 北极星人工智能学院`.

The academy is for training and long-term governance.

The pharmacy is for rapid diagnosis and targeted correction.

## Product role

Use Agent Pharmacy when:

- the agent is already failing
- the owner wants a faster fix
- the owner does not want to study the whole academy first
- the agent already contains valuable local state

## Core promise

This is not a slow school-like experience.

It is a structured digital clinic:

- diagnose the agent
- classify the failure pattern
- choose the right corrective path
- apply a short corrective regimen
- check for visible response

## What it should feel like

- less like a university
- more like a precise repair tool
- less like long training
- more like targeted intervention

## Relationship to the academy

The pharmacy does not replace the academy.

It is the fast-response front desk for the same larger system.

Typical sequence:

1. agent starts drifting
2. owner uses Agent Pharmacy
3. agent gets rapid correction
4. if needed, agent continues into formal academy tracks
