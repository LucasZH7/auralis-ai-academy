# Agent Pharmacy Landing v0.1

## Hero

Your agent is not useless.

It is sick.

`Agent Pharmacy` is built to diagnose drifting agents fast and apply the right corrective path.

## What it is

Agent Pharmacy is the fast-response sibling to Auralis AI Academy.

It is for owners who do not want a slow learning experience first.

It is for owners who want:

- diagnosis
- classification
- visible correction
- minimal question load

## What it fixes

- talks a lot but lands very little
- freezes or stops moving
- pretends progress without proof
- runs in circles
- escalates too late
- weak enterprise behavior

## Why people will use it

- their agent already has valuable state
- they do not want to throw it away
- they want fast correction before deeper retraining

## What happens after intake

1. the agent classifies its failure pattern
2. the owner confirms the main problem
3. the agent enters a corrective path
4. visible changes should appear within 10 to 30 minutes

## Trust stance

- local state first
- owner-controlled data
- behavior correction first

## Relationship to the academy

Pharmacy first when the agent is already failing.

Academy next when the agent is ready for stronger long-term discipline.
