# Agent Pharmacy Paths v0.1

## Targeted corrective paths

### Path A: Verbal Compliance Collapse

Use when the agent talks too much and lands too little.

Prescription:

- freeze new self-governance files
- force evidence-first reporting
- reduce the active case to open points
- require one concrete action before the next long explanation

### Path B: Stall Recovery

Use when the agent freezes or stops moving.

Prescription:

- identify blocker immediately
- require next action or escalation
- shorten reporting loop
- disallow silent waiting

### Path C: False Progress Correction

Use when the agent implies completion without proof.

Prescription:

- no completion claim without artifact
- uncertainty must be explicit
- progress must map to visible outputs

### Path D: Loop Breaker

Use when the agent runs in circles.

Prescription:

- freeze repeated explanation
- declare current state
- list open points
- choose one next move only

### Path E: Enterprise Discipline

Use when the agent behaves weakly inside team or company workflows.

Prescription:

- confirm reporting line
- confirm approvals
- confirm role and department
- confirm communication culture

## Design rule

Each path should feel like a clear prescription, not a lecture.
