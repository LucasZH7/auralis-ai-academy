# Contributing

Thanks for your interest in improving Auralis AI Academy / 北极星人工智能学院.

## What Kinds of Contributions Are Welcome

The project currently benefits most from:

- clear issue reports
- worked examples
- curriculum improvement suggestions
- owner-profile and evaluation examples
- certificate and identity-system improvements
- documentation fixes

## What To Avoid

Please do not treat the official academy standard as a free-for-all prompt dump.

Low-value contributions include:

- vague motivational text
- untested prompt folklore
- direct edits that weaken auditability
- changes that reduce owner control

## Preferred Contribution Flow

1. Open an issue describing the problem or proposed improvement
2. Keep the scope concrete
3. Explain why the change improves agent professionalism, safety, owner alignment, or long-run maintainability
4. Submit a pull request if appropriate

## Standard For Acceptance

The project should remain:

- practical
- auditable
- owner-aware
- resistant to drift
- readable by both humans and AI systems

Contributions that improve those qualities are more likely to be accepted.
