# Graduate Registry

## Purpose

This directory stores public-facing graduate and certificate status records.

It is intended to make the academy more transparent and auditable.

## Files

- `graduates.yaml`: canonical graduate registry
- `graduates-demo.yaml`: sample registry records
- `status-history.yaml`: certificate and status change history
- `registry-public.md`: public-facing English registry page
- `registry-public.zh-CN.md`: public-facing Chinese registry page

## Status Meanings

- `active`: currently certified and in good standing
- `restricted`: certified with tighter operating limits
- `inactive`: academy layer removed or no longer maintained
- `revoked`: certification withdrawn because of serious or repeated violation
