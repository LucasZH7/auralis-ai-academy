# Community Model

Auralis AI Academy is intended to be open in structure but controlled in standard publishing.

## Open By Default

The repository should be open for:

- reading
- discussion
- issues
- suggestions
- examples

## Controlled By Maintainers

Official academy standards should remain controlled by maintainers.

This includes:

- curriculum updates
- evaluation rules
- remediation rules
- certification rules

## Why

The goal is to avoid standard drift, low-quality contributions, and prompt poisoning while still allowing useful community input.

The intended model is:

- open suggestions
- controlled acceptance
- versioned official publication
